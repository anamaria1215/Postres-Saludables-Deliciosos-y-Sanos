# Postres-Saludables-Deliciosos-y-Sanos
API Backend desarrollada en NestJS para el proyecto “Delicioso y Sano”, una plataforma dedicada a la venta de postres saludables. Incluye autenticación JWT, CRUD completo, conexión a PostgreSQL, manejo de roles, documentación con Swagger, pruebas unitarias y despliegue en Railway.
